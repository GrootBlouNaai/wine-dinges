Эта директория необходима для работы скрипта. Как минимум никогда не
удаляйте файл game_info.txt и каталог data.

Остальные каталоги и файлы можно удалить или не создавать,
если вы не собираетесь их использовать.

Описание директорий/файлов:

game_info.txt - информация об игре
data - каталог с файлами игры
dlls - библиотеки и прочие файлы, которые нужно поместить в system32 или syswow64
additional - специфические файлы (например, настройки для игры или шрифты)
regs - твики для реестра
exe - исполняемые файлы (чаще всего, установщики разных программ)

WINETRICKS:

winetricks_list.txt - список компонентов для установки через winetricks.

При создании префикса winetricks автоматически установит все компоненты
и применит все твики, написанные в файле winetricks_list.txt. Нужно писать
все в первой строке, разделяя элементы пробелом.

Например: d3dx9 corefonts xact dxvk win8.

Если сам Winetricks не найден в системе, то скрипт автоматически скачает
его.

Описание каталогов:

========================= Каталог data ================================

Здесь должны находиться файлы игры.

========================= Каталог dlls ================================

Здесь должны находится файлы, которые нужно поместить в каталог
windows/system32 при создании префикса.

Если вы хотите конкретно указать каталог, куда нужно класть файлы
(например, system32 или syswow64), тогда кладите их по пути
game_info/dlls/system32 или game_info/dlls/syswow64.

Скрипт также автоматически переопределит эти файлы на
"Сторонние" в winecfg и зарегистрирует с помощью regsvr32.
В основном, сюда нужно класть dll файлы, но можно также класть файлы
с любыми другими расширениями.

============================ Каталог exe ==============================

Здесь должны находиться исполняемые файлы (например, установочники).

Скрипт автоматически выполнит все файлы из этого каталога
при создании префикса.

============================ Каталог regs =============================

Здесь должны находиться reg файлы.

Скрипт автоматически импортирует все reg файлы из этого каталога
при создании префикса.

======================== Каталог additional ===========================

Специфичные для игр настройки и прочие файлы. Класть в директории
соответствующие реальным названиям директорий. Например:

Содержимое каталога game_info/additional/prefix/drive_c/Windows/Fonts будет скопировано
при создании префикса в каталог WINEPREFIX/drive_c/Windows/Fonts.

Содержимое каталога game_info/additional/documents/Documents_Multilocale/My Games/Fallout4
будет скопировано в каталог DOCUMENTS_DIR/Documents_Multilocale/My Games/Fallout4.

И т.д.
